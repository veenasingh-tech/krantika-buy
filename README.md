# 🛒 Krantika Buy - E-commerce Landing Page
Krantika Buy is a modern and responsive e-commerce landing page built to provide a seamless shopping experience.

### 🚀 Features:
- **Responsive Design:** Works perfectly on mobile, tablet, and desktop.
- **Interactive UI:** Smooth navigation and product hover effects.
- **Dynamic Elements:** Simple JavaScript integration for better user interaction.

### 🛠️ Tech Stack:
- **HTML** for structure.
- **CSS** for styling and animations.
- **JavaScript** for basic interactivity.
